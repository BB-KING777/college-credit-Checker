単位チェッカーとは？

このアプリケーションは、大学生が自分の取得単位を簡単に管理・確認できるツールです。卒業に必要な単位数や履修状況を視覚的に把握することができます。

単位の確認

取得した単位を科目区分ごとに確認できます

視覚的な表示

グラフで履修状況を一目で把握

卒業シミュレーション

卒業までに必要な残りの単位数を計算

アプリを試す

アプリを起動

デモボタンをクリックするとアプリのデモが表示されます

使い方

  1. 「アプリを起動」ボタンをクリックしてアプリを開きます
  2. 科目区分と取得単位数を入力します
  3. 「計算」ボタンをクリックして結果を確認します
  4. 必要に応じてデータを保存・共有できます
